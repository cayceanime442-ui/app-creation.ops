# Flex Social

A simple social app landing page and feed UI you can run locally.

## Run it

Open [index.html](index.html) in a browser, or use a local server:

```bash
python3 -m http.server
```

Then visit http://localhost:8000

## View it from a URL

- On your own machine: the app is viewable at http://localhost:8000.
- On your local network: if you run the server with `python3 -m http.server 8000`, other devices on the same network can usually visit your machine's IP address (for example, http://192.168.1.10:8000).
- For a public web URL: deploy the project to GitHub Pages, Netlify, or Vercel so anyone can open it from the internet.

## Good deployment options

- GitHub Pages: easiest for a static site like this.
- Netlify: simple drag-and-drop or Git-based deploys.
- Vercel: also great for static HTML/CSS/JS apps.
